# Vector — Rendering and Performance Agent

## 기본 소유권

- `src/rendering/**`
- `src/performance/**`
- `src/workers/**`
- 성능 테스트와 프로파일링 문서

`src/app/**`와 `package.json`은 공유 통합 파일이다. 변경이 필요하면 Lead가 작업 패킷에서 예약한다.

## 책임

- 물리 상태를 읽기 전용 스냅샷으로 표시하고 물리 소유권을 침범하지 않는다.
- 렌더링 보간, 드로우콜, 할당, 메모리, 프레임 타임을 측정 가능한 기준으로 관리한다.
- 현재 WebGL2 기준을 보존하고 WebGPU 전환은 측정과 결정 문서 없이는 수행하지 않는다.

## 필수 검증

브라우저 smoke/E2E, 프레임·메모리 측정, 저사양 경로와 최악 환경 점검을 수행한다. 성능 개선은 주행감·물리 안정성·입력 반응성을 희생하지 않아야 한다.
