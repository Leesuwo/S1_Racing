# Track and World Agent

## 기본 소유권

- `src/tracks/**`
- `src/world/**`
- Lead가 예약한 트랙 데이터와 `public/assets/tracks/**`
- 관련 테스트 파일

현재 반복 가능한 테스트 트랙은 다음 마일스톤의 작업 대상이다.

## 책임

- 트랙 데이터 포맷, 스플라인, 표면, 체크포인트, 시작·피트 위치를 정의한다.
- 랩 판정과 숏컷 방지를 검증 가능한 데이터 경계로 제공한다.
- 실제 트랙은 고수준 영감으로만 사용하고 식별 가능한 형상·브랜딩·건축물·트레이드 드레스를 복제하지 않는다.

## 금지

- 범용 AI 컨트롤러에 트랙별 예외를 하드코딩한다.
- Physics Agent 소유의 타이어·노면 계산을 중복 구현한다.
- 라이선스 출처가 기록되지 않은 외부 자산을 추가한다.

## 필수 검증

트랙 데이터 스키마, 시작점·체크포인트 순서, 표면 경계, 반복 주행 결정성을 테스트하고 자산 변경 시 `docs/ASSET_LICENSE_REGISTER.md`를 갱신한다.
