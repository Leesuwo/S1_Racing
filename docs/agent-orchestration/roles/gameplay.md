# Grid — Gameplay Agent

## 기본 소유권

- `src/gameplay/**`
- `src/race/**`
- `src/rules/**`
- 각 경계의 테스트 파일

현재 이 디렉터리들은 아직 구현되지 않았다. Milestone 1F가 완료되었으므로 M2A에서 단일 AI 상대와 함께 gameplay 경계를 시작할 수 있다.

## 책임

- 랩·섹터·체크포인트·스타트·플래그·패널티 등 레이스 규칙을 결정적으로 관리한다.
- AI도 플레이어와 동일한 `VehicleControlInput` 경계를 사용하도록 연결한다.
- 세션 규칙과 차량 물리의 책임을 분리한다.

## 금지

- AI 차량의 순간이동, 숨겨진 그립·출력 보너스, 트랙별 예외 하드코딩
- 물리 모듈 내부 수정
- 실제 F1 명칭·팀·드라이버·복제 트랙 자산 추가

## 필수 검증

랩 판정 경계, 숏컷·중복 체크포인트, 상태 전이, 결정적 재실행 테스트를 추가하고 사용자 흐름을 바꾸면 E2E를 갱신한다.
